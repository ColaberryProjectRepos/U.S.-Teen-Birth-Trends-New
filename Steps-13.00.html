Creating Measures 13


There are several ways to take to create measures.

Steps:

  1. Right click on the Measures table and click on New Measure or From the Home Tab, either on Data View or Report View, Click on New Measure from the ribbon. All the options will land on the same spot.
  2. On the Formula bar, write the DAX formula for the measure. Once done press Enter or Click on commit (Checkmark icon) on the left of the formula.
  3. After Creating each measure, choose the appropriate format and decimal for each value (Measure) and click on save.


Follow the above steps to create all required measures. Below are all the listed measures together with the DAX formulas for creating the measures.

Births by Year =

SUMX(VALUES('US Teen Birth Trends'[Year]),
CALCULATE(SUM('US Teen Birth Trends'[State Births])
)
)

Total US Births = SUM('US Teen Birth Trends'[U.S. Births])

US Births LY = CALCULATE ([Total US Births], SAMEPERIODLASTYEAR('US Teen Birth Trends'[Year]))

Year Over Year measure to determine the difference between last year's and the current year's births. In this measure, you will type the following DAX code:

YoY % Change =
VAR CurrentYear = MAX('US Teen Birth Trends'[Year])
VAR PrevYear = CurrentYear - 1
RETURN
DIVIDE( SUMX(
FILTER('US Teen Birth Trends', 'US Teen Birth Trends'[Year] = CurrentYear),
'US Teen Birth Trends'[State Births]
) -
SUMX(
FILTER('US Teen Birth Trends', 'US Teen Birth Trends'[Year] = PrevYear),
'US Teen Birth Trends'[State Births]
),
SUMX(
FILTER('US Teen Birth Trends', 'US Teen Birth Trends'[Year] = PrevYear),
'US Teen Birth Trends' [State Births]
)
)


YOY Births % =
VAR CurrentYear = MAX('US Teen Birth Trends'[Year])
VAR CurrentYearBirths = CALCULATE(SUM('US Teen Birth Trends' [State Births]), 'US Teen Birth Trends' [Year] = CurrentYear)
VAR PreviousYearBirths = CALCULATE('US Teen Birth Trends'[State Births]), 'US Teen Birth Trends' [Year] = CurrentYear - 1)
RETURN IF(ISBLANK(PreviousYearBirths), BLANK(), (CurrentYearBirths - PreviousYearBirths) / PreviousYearBirths)

Arrow YOY% = SWITCH(TRUE(),[YoY % Change] <0.01,UNICHAR(9662),[YoY % Change] =0.10, UNICHAR(9666), UNICHAR(9652))
