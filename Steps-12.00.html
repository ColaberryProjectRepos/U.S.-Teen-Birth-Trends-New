Creating a Measures Table 12


Steps:

  1. From Data View or Report View, Go to Home tab
  2. Click on Enter Data,
  3. Give the Table a name "Measures Table".
  4. Click on Load.

A new table named Measures Table appears on the Data Pane.




One of the most common ways to organize the data model is to create a separate table exclusively for your measures. This is so that you don't have to feel like you are getting lost in a sea of tables and fields whenever you open the data.
