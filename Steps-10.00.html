Transforming the Data in Power Query Editor 10


Steps:

  1. Here, make sure that you familiarize yourself with the data, clean and transform it. On the Year Column Title click on the down arrow to filter the years so that you are only working with rows that contain years 2008 to 2018. (This is to ensure we only load the data that you need for this project).
  2. Rename The Table from NCHS-US and State Trends .... to US Teen Birth Trends, Right click on the table name and rename.
  3. Remove all the rows that contain (Total U.S.) as we have a column that contains the Total U.S.
  4. Click on the down arrow for the column that has Age Group years to filter the rows, make sure have rows that contain ages 15-17 and 18-19 age group years.
  5. Round off the Columns that have the State Rate and the U.S. Birth Rate to have a whole number instead of a decimal.
