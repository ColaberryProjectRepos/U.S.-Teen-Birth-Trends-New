Creating the Waterfall Chart 26



Steps:
  1. Click on an empty space on the canvas,
  2. From the Visualization pane Select Waterfall Chart.
  3. Fill in the value fields provided below for the necessary fields. (Age Group (Years), Births by Year, Total US Births, YOY Births %)
Build Visual:
Category = Age Group (Years)
Y-axis = YOY Births %
Tooltips = Births by Year, Total US Births

Format Visual:
X-axis = On, Values: Font = Segoe UI, Size = 10, Color =#605E5C, Title = Off
Y-axis = On, Values: Font = Segoe UI, Size = 10, Color =#605E5C, Title = Off
Legend = On, Options = Top Center, Font = Segoe UI, Font Size = 10, Color = #605E5C
Columns: Colors = Increase = #DF3A11, Decrease = #1AAB40,
Total column = On, Total = # E66C37
Data labels = On

General
