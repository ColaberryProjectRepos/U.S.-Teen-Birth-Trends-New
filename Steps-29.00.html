Creating the Map Legend 29



Steps:
  1. On Report View, Go to Insert Tab,
  2. Click on Image on the Ribbon
  3. Select the Legend from the “Legend Folder” and click open.
  4. Format the Legend as follows:
General 
Height = 50
Width = 130
Horizontal = 1029
Vertical = 423


     5. Add two text boxes, one to the right and the other to the left of the Legend.
     6. Format the text boxes as follows: 

Text box 1
Height = 28
Width = 46
Horizontal = 983
Vertical = 432

Type Low in the text box, Font = Segoe UI, Font Size = 10, Color = #252423
Background = On, Color = E66C37, Transparency = 90%

Text box 2
Height = 28
Width = 51
Horizontal = 1159
Vertical = 432

Type High in the text box, Font = Segoe UI, Font Size = 10, Color = #252423
Background = On, Color = E66C37, Transparency = 90%
